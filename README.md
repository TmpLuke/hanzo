# Hanzo Marketplace

Modern marketplace frontend with admin panel, product management, verified MoneyMotion revenue, and live UI.

## Key Features
- Admin dashboard with multi-series revenue chart (revenue, orders, AOV, forecast)
- MoneyMotion session verification for accurate revenue
- Products, variants, coupons, and orders management
- Live announcement banner and trending products

## Dev
```bash
npm install
npm run dev
```
